Popover

Popover is an generic element that displays content in a floating window

Showcase
PREVIEW HTML TS CSS
Source code
API

PopoverComponent
componentapp-popover

Inputs

NameTypeDefaultDescription
isOpen model<boolean>()false Determines whether the popover is open or closed
origin PopoverOriginDirective-- The element to which the popover should be attached
offsetX number0 The horizontal offset of the popover relative to the origin
offsetY number0 The vertical offset of the popover relative to the origin
styled booleanfalse Applies default styles to the popover
hasBackdrop booleanfalse Displays a backdrop behind the popover
blockScroll booleanfalse Prevents scrolling of the page when popover is open
outsideClose booleanfalse Closes the popover when the user clicks outside or presses the escape key
stretchToOrigin booleanfalse Stretches the width of the popover to fit the origin element
restoreFocus booleanfalse Restores focus to the origin element after the popover is closed
positionY top | right | bottom | leftbottom The vertical position of the popover relative to the origin

Outputs

NameValueDescription
opened void Emits when the popover is opened
closed void Emits when the popover is closed

PopoverOriginDirective
directive[appPopoverOrigin]

The element to which the popover should be attached
