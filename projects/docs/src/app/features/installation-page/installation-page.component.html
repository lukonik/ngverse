Automatic Installation (Recommended)

NgVerse is not an installable library in the traditional sense; there's no need to run npm install ngverse. Instead, NgVerse provides a convenient schematic to handle the setup for you, making the process of copying and integrating components into your project seamless and automatic. To get started, run the following command:

What Does This Do?

  • Adds NgVerse schematics dependency: This will manage the integration of components into your project without the need to manually copy or paste them.
  • Add Angular/cdk: Provides utility features like keyboard navigation and floating panel support (e.g., dialogs), which are essential for certain NgVerse components.
  • Provide Animations config: Adds animation configurations to the root of your project for seamless functionality of components like the accordion, which relies on smooth open/close animations.
  • Add globals.scss style file: globals.scss file will be added in your project and imported into your styles.scss. For more details, refer to the Theming Guide.
