The minimum required Angular version for installation is >=19.0.2.

Relies on following third-party libraries to get maximum flexibility and performance

  • @angular/cdk
  • Tailwind CSS
  • @ngverse/motion

Install @angular/cdk

leverages Angular CDK as a low-level toolkit to handle essential functionalities such as accessibility, floating panels, and other foundational UI behaviors.

import @angular/cdk styles to your styles.css file

Install tailwind (v4)

uses Tailwind CSS as a CSS framework

Create a .postcssrc.json file in the root of your project and add the @tailwindcss/postcss plugin to your PostCSS configuration.

Add an @import to styles.css that imports Tailwind CSS.

Add Angular animations

In order to do smooth transitions between states, we use @ngverse/motion library.
