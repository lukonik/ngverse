To add a component to your project, run the following command: sh Copy Edit

This will generate a button folder in your project's root directory, containing the ButtonComponent. Additionally, the generator provides the following options:

Options

1. path (Optional)

Specifies where to place the component folder.

  • By default, the component is added to the root folder (/src/app).
  • You can specify a different location using the --path parameter. Example: ng g ngverse:component button --path=core This will place the component in /src/app/core.

2. prefix (Optional)

Specifies the prefix of component

  • By default, it will use the project's prefix. If no prefix is provided, it will default to 'app'.

3. replace (Optional, default: false)

Determines whether to replace existing files with the same name.

  • If a folder with the same name as the component already exists, the schematic will throw an error to prevent accidental file deletions.
  • If you want to override the existing files, set replace=true.
  • Example: ng g ngverse:component button --replace=true

4. project (Optional)

The name of the Angular project where the component should be added.

  • If your workspace has only one project, the component will be added automatically without needing this parameter.
  • If your workspace contains multiple projects, you can specify which one to use by passing the project name.
  • By default, the component is added to the default project specified in angular.json.
