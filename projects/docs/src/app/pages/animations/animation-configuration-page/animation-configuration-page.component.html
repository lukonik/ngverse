Animations module includes collection of animations build with@angular/animations.

Schematic Usage

Options

  • path: Specifies where to place the animation.
  • project: The name of the Angular project where the animation should be added. By default it will use defaultProject
  • replace: Determines whether to replace existing files with the same name. False by Default
