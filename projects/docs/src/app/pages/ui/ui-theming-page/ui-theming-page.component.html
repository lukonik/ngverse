UI module comes with a predefined Tailwind CSS theme configuration, that is shared across the components. It includes light and dark mode. copy the styles and paste it into your styles.css

For example, by default, the primary color is: --color-blue-500 If you want to change it to amber, simply update it to: --color-amber-500 This makes customization easy and ensures design consistency across your project.

CSS Variables follow the following pattern:

  • --color-name: Background color
  • --color-name-foreground: Text color

These styles are meant to help you get started quickly with beautiful components that share a consistent look and feel. Your designers can explore countless ways to customize the design—such as updating color tokens and more. Don’t hesitate to modify the component styles directly to suit your needs.
