ngverse:ui Schematic is used to generate UI components in your app.

For example: ng g ngverse:ui button will generate a button component.

Schematic Options

  • path: Specifies where to place the component folder.
  • project: The name of the Angular project where the component should be added. By default it will use defaultProject
  • prefix: Specifies the prefix of component. By default, it will use the project's prefix, or 'app' if no project is provided.
  • includeTests: Specifies whether to include test files for the element. False by Default
  • replace: If component already exists, overwrite it. False by Default
